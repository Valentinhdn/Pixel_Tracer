# Pixel Tracer
## Compiler et exécuter

```$ gcc pixel_tracer.c && ./a.out```

## Info utilisation

Toutes les commandes et les formes s'écrivent en MAJUSCULE. 
Pour afficher les commandes générales, tapez HELP.
Pour afficher toutes les formes, tapez SHAPE.

## Utilité des fonctions

La fonction 'void execute_command' permet d'attribuer à chaque chaîne de caractère une commande spécifique.
La fonction 'void delete_shape' permet de supprimer des formes de la liste.
Les fonctions 'create' permettent de créer les formes souhaités par l'utilisateur. 
La fonction main() permet d'entrer les commandes et fait appelle à la fonction 'execute_commande'.
